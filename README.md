# apache-tez-dist

![](https://github.com/mpolatcan/apache-tez-dist/workflows/tez-release/badge.svg)

**Github Actions** workflow that builds **Apache Tez** for each **Apache Hadoop**
versions and releases them.

## Versions

- 0.9.2-hadoop-2.10.1
- 0.9.2-hadoop-2.9.2
- 0.9.2-hadoop-2.8.5
- 0.8.5-hadoop-2.9.2
- 0.8.5-hadoop-2.8.5
- 0.7.1-hadoop-2.9.2
- 0.7.1-hadoop-2.8.5
